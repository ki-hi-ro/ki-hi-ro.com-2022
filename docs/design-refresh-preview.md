# Design refresh preview

MAMPを起動した状態で、以下のURLから3案を切り替えて確認できます。

- Editorial: <http://localhost:8888/?design=editorial>
- Grid: <http://localhost:8888/?design=grid>
- Night: <http://localhost:8888/?design=night>

画面右下（スマートフォンでは画面下部）の `Design` スイッチからも切り替えられます。選択した案はCookieに保存されるため、記事詳細やアーカイブへ移動しても同じデザインが維持されます。

## Editorial streams

ヘッダーとフッターから、以下の3つの記事分類を確認できます。

- 技術ブログ: <http://localhost:8888/?topic=technology>
- 人生哲学: <http://localhost:8888/?topic=philosophy>
- 情報のセレクトショップ: <http://localhost:8888/?topic=curation>

既存記事のデータを破壊的に書き換えず、タグから3分類を生成しています。「情報のセレクトショップ」タグを最優先し、次に技術系タグ、どちらにも該当しない記事を人生哲学として扱うため、同じ記事が複数分類へ重複しません。

スマートフォンの記事一覧は最初の5件を表示し、必要な場合は「さらに5件を見る」で続きを展開できます。記事詳細では上下にパンくずを表示し、右下の目次ボタンで目次と元の位置を往復できます。

## Concepts

### 01 Editorial

生成りの紙面、明朝体、大きな余白、朱色を組み合わせた落ち着いたエディトリアル案です。文章を主役にしつつ、個人メディアらしい温度を残しています。

### 02 Grid

黒・白・エレクトリックブルー・ライムで構成した、シャープで実験的な案です。直線、グリッド、太い見出し、強い影でWeb制作者らしさを前面に出しています。

### 03 Night

深いネイビーをベースに、コーラルとミントの光を重ねたダーク案です。カードには透明感を持たせ、静かで没入感のある印象にしています。
